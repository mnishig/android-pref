# this folder is pref. sample

## Overview
Android Studio の SettingsActivity の Wizard で作成したものを
General Preference だけに絞り他の Fragment は削除し、
さらに、2階層あったメニュー構造を1階層のみにしたもの

This project is very simple prefernce sample.
Base of project code generate with Android Studio's wizard that I choose "Settings Activity" .
and modifying followings:

- delete preference hierarchy (top level is delete)
- generalPreference is set to top level
- other preferences delete
- unuse resource string delete
- toolbar delete
